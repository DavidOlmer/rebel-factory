# Rebel AI Factory

## Mission
Enterprise AI agent platform for Rebelgroup.com - enabling consultants to deploy, manage, and learn from AI agents.

## Features
- 🤖 Agent Management (create, configure, monitor)
- 📋 Template Library (sector-specific templates)
- 💬 Prompt Library (versioned, rated prompts)
- 📊 Telemetry Dashboard (runs, quality, costs)
- 🔐 Microsoft SSO (Azure AD/Entra ID)
- 🏢 Multi-tenant (sector isolation)
- ✅ Approval Workflows (agent promotion)
- 📈 Self-learning (drift detection, insights)

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Express + TypeScript
- Database: PostgreSQL
- Auth: Azure AD / Entra ID
- Hosting: Render.com

## Agent Tiers
1. **Personal** - Individual consultant agents
2. **Venture** - Team/sector shared agents
3. **Core** - Organization-wide approved agents
